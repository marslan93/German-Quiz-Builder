# 🇩🇪 Deutsch Wortschatz A1–B1 (DTZ)

An interactive German vocabulary trainer designed for learners preparing for DTZ and Goethe A1–B1 exams.

Eine interaktive App zum Lernen von deutschem Wortschatz für DTZ- und Goethe-Prüfungen (A1–B1).

---

## 🚀 Features

- A1 – A2 – B1 level-based vocabulary  
- Real exam-oriented words (Goethe / DTZ)  
- Quiz-based learning system  
- Visual feedback:
  - ✅ Correct answer → green screen animation  
  - ❌ Wrong answer → red screen animation  
- Favorites system for difficult words  
- Mobile-friendly modern UI  
- Fast and smooth experience  

---

## 📚 Content

### A1 – Beginner
Basic vocabulary: daily objects, food, family, colors  

### A2 – Elementary
Professions, city life, transportation, common verbs  

### B1 – Intermediate
Abstract concepts, work life, separable & reflexive verbs  

---

## 🎯 Purpose

This app is built to help learners actively practice vocabulary instead of memorizing passively.

Ideal for:
- DTZ (Deutsch-Test für Zuwanderer)
- Goethe-Zertifikat A1–B1
